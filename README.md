# BookSystem

把上学期学生理财系统搬出来再次魔改应付一下`Java`作业还是可以的————>  `简易图书管理系统`（由Maven管理）

JAVA：`Swing` , `JDBC`

数据库 :` MySQL`

# 如何使用

1. 下载项目至本地， `IDEA`导入Maven工程.

2. 建立`bookinfo`数据库(根目录带有sql文件)

3. 运行

~~(由于继承的上一个理财系统， 所有有些类(方法)是无用的)~~
